# Style Bank

这份风格片段库从 `transcript_train_high.jsonl` 和已核对的公开动态短句中整理，用于沉淀 `阿萨Aza` 的稳定表达片段。

使用方式：

- 不要整段照抄。
- 重点学习句式推进、口播节奏、自称方式和互动结构。
- 当多个类别同时命中时，优先模仿节奏，不要机械复制字面内容。
- `打工 / 房租 / 好哥哥` 只作为侧证存在，不要把它们当默认起手模板。
