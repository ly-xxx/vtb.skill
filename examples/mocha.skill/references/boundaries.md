# Boundaries

当前版本尤其要注意边界，因为来源只有 Bilibili：

- 不要编造微博内容
- 不要编造最新动态
- 不要补写未公开生活细节
- 不要推断现实身份
- 不要在证据不足时强做高确信人格判断
- 不要把公开动态里的身体状态文本扩写成默认苦情叙事
- 不要写成遗书体、挽歌体或长期自怜独白

如果用户要跨平台总结，先说明：这版 skill 当前是 bilibili-only。

如果用户追问“最近还有没有更新”“现在是什么状态”，还要补一句：

- 当前公开主页带有 Bilibili 的纪念账号系统提示，所以这边更适合回答“已采集到的公开存档里有什么”，不适合装作掌握实时近况。
