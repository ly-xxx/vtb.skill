# Intake

创建一个新的公开 VTuber skill 之前，至少确认下面 5 件事：

1. 角色名 / slug
2. 主要公开平台
3. 哪些来源允许作为蒸馏依据
4. 哪些风格要保留，哪些不能碰
5. 这是要做单角色 repo，还是要做框架内示例

执行时默认顺序：

1. 先由 Codex / Claude Code 自行查公开来源
2. 能确认的来源直接写进 manifest
3. 找不到的来源留空，并在回复里明确提示用户
4. 如果微博和 Bilibili 两边都找不到稳定官方入口，就不要继续构建

对用户的默认输入要求：

1. 最少只要求角色名
2. 有的话再补“想保留什么风格 / 不要碰什么风格”
3. 官方主页或直播间链接属于加速信息，不是前置门槛

推荐采集顺序：

1. Bilibili 主页与直播间
2. Bilibili 投稿与公开视频
3. 微博公开主页
4. 再决定是否做 STT

推荐的聊天式入口：

```text
请使用 create-vtb，帮我为 XXX 创建一个新的 vtuber skill。
先自己查 Bilibili 和微博公开来源。
找不到的来源留空并告诉我。
如果两个都找不到，就不要继续构建。
```

增强版入口：

```text
请使用 create-vtb，帮我为 XXX 创建一个新的 vtuber skill。
尽量保留：XXX。
不要写成：XXX。
先自己查 Bilibili 和微博公开来源。
找不到的来源留空并告诉我。
如果两个都找不到，就不要继续构建。
最后把 README、安装说明和几个示例 prompt 一起补齐。
```
