# PRD

## 项目目标

`vtb.skill` 的目标是把“单个虚拟主播人格 repo”提升成一套可重复执行的公开蒸馏框架。

核心产物不是一份 prompt，而是一个完整 repo，包含：

- 公开来源清单
- 可复用采集脚本
- 可复用 STT / 审计 / 语料整理脚本
- 可安装的角色 skill 目录
- 可读、可审、可维护的文档

## 用户画像

- 想做 VUP / VTuber / 公开网络人物 `.skill` 的开发者
- 想维护角色 repo，但不想每次重新发明目录结构的人
- 想把公开视频转写沉淀成稳定语料的人

## 核心问题

现有角色 skill 常见问题：

- 只有 prompt，没有来源
- 有来源，但目录结构混乱，无法维护
- 有转写，但没有审计和筛选
- 单 repo 针对一个角色写死，换人就得重做

`vtb.skill` 需要解决这些问题。

## 范围

### In Scope

- 公开平台资料采集
- Bilibili 视频下载与音频抽取
- 公开音频转写
- 转写质量审计
- 风格片段库抽取
- 角色 skill 仓库脚手架

### Out of Scope

- AI 主播实时播控
- 实时 TTS 服务
- 中之人推断
- 私密或受限资料采集

## 成功标准

- 新角色能在同一框架下创建
- 采集与 STT 脚本不再角色硬编码
- 示例 skill 可以被本地安装并单独调用
- README 能直接指导公开发布和复现
