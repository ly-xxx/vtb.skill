# Source Playbook

## 推荐顺序

1. 先抓 Bilibili 主页与直播间信息
2. 再抓公开视频列表
3. 最后再看微博是否值得补

## Bilibili 公开源价值排序

1. 直播间简介 / 公告
2. 自我介绍视频
3. 明显口播向投稿
4. 普通投稿简介
5. 评论区

## 微博使用建议

- 主要用来补动态文案和短句风格
- 遇到访客墙或抓取不稳定时，不要伪造结果
- 可以把 handle、主页链接先记录在 manifest 中，等待后续补采

## 单信源规则

- 只找到 Bilibili：照样继续构建，微博留空并说明“当前未检到稳定官方微博主页”
- 只找到微博：也可以继续构建，但 STT 和视频语料路线会受限
- 微博和 Bilibili 都找不到：停止构建，不要拿二手搬运源硬凑
